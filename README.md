# ArtyrTask 🚀

Система управления задачами в стиле Trello, разработанная с использованием современного стека технологий. ArtyrTask позволяет легко организовывать работу, создавать доски, списки и карточки, а также управлять ими с помощью удобного Drag-and-Drop интерфейса.

## ✨ Основные возможности

- 🔐 **Аутентификация и Авторизация**: Безопасная регистрация и вход для пользователей (JWT токены).
- 🏢 **Рабочие пространства**: Группируйте ваши проекты по рабочим пространствам.
- 📋 **Доски и Списки**: Создавайте неограниченное количество досок и колонок (списков) внутри них.
- ✅ **Задачи (Карточки)**: Подробные карточки задач с описаниями, чекбоксами выполнения и файловыми вложениями.
- 🖱️ **Drag-and-Drop**: Интуитивно понятное перемещение задач между колонками и изменение порядка списков.
- 📎 **Вложения и Ссылки**: Прикрепляйте файлы напрямую к задачам и сохраняйте важные ссылки.
- 🌙 **Современный UI**: Красивый темный интерфейс с эффектами "стекла" (Glassmorphism), полностью переведенный на русский язык.

## 🛠️ Технологический стек

### Бэкенд (Backend)
- **Python 3.12+**
- **FastAPI** — высокопроизводительный современный веб-фреймворк.
- **SQLAlchemy** — ORM для удобной работы с базой данных (SQLite).
- **Pydantic** — валидация данных.
- **Bcrypt / JWT** — безопасное шифрование паролей и аутентификация.

### Фронтенд (Frontend)
- **Vue 3** (Composition API) — прогрессивный JavaScript-фреймворк.
- **Vite** — невероятно быстрая сборка фронтенда.
- **Pinia** — современное управление состоянием (State Management).
- **Element Plus** — богатая библиотека UI-компонентов.
- **VueDraggable** — мощный инструмент для реализации Drag-and-Drop.

## 🚀 Установка и запуск

Для запуска проекта локально вам понадобятся **Python** и **Node.js**.

### 1. Клонирование репозитория
```bash
git clone https://github.com/your-username/ArtyrTask.git
cd ArtyrTask
```

### 2. Запуск Бэкенда (API)

Перейдите в папку `backend`, установите зависимости и запустите сервер:

```bash
cd backend
python -m venv venv
# Активация виртуального окружения:
# Для Windows:
venv\Scripts\activate
# Для Linux/Mac:
# source venv/bin/activate

pip install -r requirements.txt
uvicorn main:app --reload
```
Бэкенд будет доступен по адресу: `http://localhost:8000`  
Интерактивная документация API (Swagger): `http://localhost:8000/docs`

### 3. Запуск Фронтенда

В новом окне терминала перейдите в папку `frontend`:

```bash
cd frontend
npm install
npm run dev
```
Фронтенд будет доступен по адресу: `http://localhost:5173` (порт может отличаться, смотрите вывод консоли).

## 📂 Структура проекта

```text
ArtyrTask/
├── backend/            # Исходный код API (FastAPI)
│   ├── routers/        # Маршруты (workspaces, boards, lists, tasks)
│   ├── database.py     # Настройка подключения к БД
│   ├── models.py       # SQLAlchemy модели таблиц
│   ├── schemas.py      # Pydantic схемы валидации
│   └── main.py         # Точка входа приложения
│
└── frontend/           # Исходный код клиентской части (Vue 3 + Vite)
    ├── src/
    │   ├── components/ # Переиспользуемые Vue-компоненты (BoardList, TaskCard)
    │   ├── views/      # Страницы приложения (Login, BoardView, Workspaces)
    │   ├── stores/     # Pinia-сторы для работы с API
    │   ├── services/   # Axios-клиент для отправки запросов к API
    │   └── App.vue     # Главный компонент
    └── index.html
```
